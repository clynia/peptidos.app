# peptidos.app

Portal divulgativo en español sobre péptidos: información clara y con fuentes fiables, pros y contras, y una calculadora de reconstitución pensada como demostración educativa del cálculo.

Sitio estático (HTML + CSS + JavaScript, sin paso de build).

> Contenido exclusivamente divulgativo e informativo. No es consejo médico, farmacéutico ni veterinario, ni una invitación a consumir o usar ninguna sustancia. No se vende nada desde este sitio.

## Estructura

- `index.html` — portada
- `articulos/` — fichas de cada péptido
- `herramientas/` — calculadora de reconstitución
- `paginas/` — guías y páginas legales (aviso legal, términos, privacidad)
- `assets/` — CSS, JavaScript y logotipos

## Ramas

- `main` — versión estable
- `dev` — desarrollo
